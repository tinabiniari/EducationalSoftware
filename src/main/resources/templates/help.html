Help Page

Java Educational Software is an online software,where you can learn the basic fundamentals in Java by reading our courses and taking our online tests.
As long as you are registered in our system, you can start learning Java, by clicking List of Java Modules,choose one of the modules by clicking on List of Courses and choose one of the available courses.
At the end of every course, there is a test, so you can check what you learned.If you pass this test, the next course will be unlocked and you will be able to continue with the next course. You cannot continue, if you do not pass this test.At the end of every module, you will be redirected to the test of this module.
In the section Tests, you can take a test for every module.Every test contains 10 questions. If you score 50% or more, you pass the test.If you score less than 50%, you don't pass the test and you have to study again and unlock again the courses of this module.
In the main page, you can check in the progress bar how many courses you have unlocked/passed and how many tests you have succesfuly score. Also, you can check it in the section Progress.
By clicking on the section Progress from the main Page, or the navigation bar, you can see your progress.
